平台核心特色

让调研工作变得简单高效

零代码设计

完全可视化的拖拽操作，无需编程知识，任何人都能快速创建专业问卷。

全端适配

完美支持PC、平板、手机等多终端，响应式布局，随时随地填写。

报告导出

一键生成PDF/Excel分析报告，支持自定义模板，满足汇报需求。

功能模块一览

覆盖调研全流程的完整功能矩阵

问卷设计器

  • 拖拽式编辑：左侧题型面板，中间画布，右侧属性设置
  • 逻辑跳转：根据答案动态显示/隐藏题目，实现个性化路径
  • 条件分支：支持IF-ELSE逻辑，复杂场景轻松应对
  • 题目分组：支持分页、分节，长问卷结构化管理
  • 主题样式：多种精美模板，支持自定义品牌配色

丰富题型库

  • 基础题型：单选题、多选题、填空题、下拉选择
  • 进阶题型：评分题（星级/数字/表情）、排序题、矩阵题
  • 特殊题型：文件上传、图片选择、签名手写、日期时间
  • 高级题型：NPS净推荐值、量表题、关联题
  • 自定义题型：支持HTML自定义，扩展无限可能

数据统计分析

  • ECharts图表：饼图、柱状图、折线图、雷达图、词云图
  • 交叉分析：多维度数据透视，发现隐藏规律
  • 实时统计：数据实时更新，无需手动刷新
  • 文本分析：开放式问题关键词提取、情感分析
  • 漏斗分析：转化率追踪，优化问卷完成率

权限与安全

  • 匿名模式：浏览器指纹识别，保护用户隐私
  • 实名模式：Token认证，精准识别用户身份
  • 访问控制：密码保护、IP限制、时间窗口
  • 防刷机制：验证码、频率限制、异常检测
  • 数据加密：传输加密、存储脱敏，符合GDPR规范

快速开始

三步启动您的问卷调研平台

Step 1: 获取代码

从GitHub或Gitee克隆项目代码到本地

git clone https://github.com/zhaoyachao/zdh_web.git

Step 2: 编译打包

使用Maven进行项目编译和打包

sh build.sh

Step 3: 启动应用

启动应用并访问问卷调研平台

sh bin/start.sh
# 访问 http://localhost:8081

配置说明

在启动前，请确保在 application.properties 中配置以下参数：

# 数据库配置
spring.datasource.url=jdbc:mysql://localhost:3306/zdh
spring.datasource.username=root
spring.datasource.password=your_password

# 文件上传配置（可选）
survey.upload.path=/data/survey/uploads
survey.upload.max-size=10MB
