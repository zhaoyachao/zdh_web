功能模块一览

覆盖微信运营全场景的完整功能矩阵

公众号管理

  • 多账号管理：支持同时管理多个微信公众号
  • 菜单配置：可视化菜单编辑器，支持自定义菜单
  • 素材管理：图文、图片、视频、语音素材统一管理
  • 自动回复：关注回复、关键词回复、默认回复
  • 草稿箱：文章编辑与预览发布

消息推送

  • 模板消息：服务通知、订单提醒、状态更新
  • 群发消息：图文群发、文本群发、定向推送
  • 客服消息：实时客服对话、消息记录查询
  • 定时发送：预设时间自动发送消息
  • A/B测试：消息内容对比测试优化

用户运营

  • 用户画像：多维度用户标签与属性分析
  • 标签系统：自定义标签、自动打标、批量管理
  • 粉丝分析：新增取关、活跃度、地域分布
  • 生命周期：新用户、活跃用户、流失预警
  • 黑名单：用户黑名单管理与导出

数据统计

  • 图文分析：阅读量、分享量、收藏量统计
  • 用户增长：日增/周增/月增趋势图
  • 消息效果：送达率、点击率、转化率
  • 转化漏斗：从关注到转化的全链路追踪
  • 自定义报表：灵活配置数据维度和指标

快速开始

三步启动您的微信智能运营平台

Step 1: 获取代码

从GitHub或Gitee克隆项目代码到本地

git clone https://github.com/zhaoyachao/zdh_web.git

Step 2: 编译打包

使用Maven进行项目编译和打包

sh build.sh

Step 3: 启动应用

启动应用并访问微信运营平台

sh bin/start.sh
# 访问 http://localhost:8081

配置说明

在启动前，请确保在 application.properties 中配置以下参数：

# 微信公众号配置
wx.appid=your_app_id
wx.appsecret=your_app_secret
wx.token=your_token
wx.aes_key=your_aes_key

# 数据库配置
spring.datasource.url=jdbc:mysql://localhost:3306/zdh
spring.datasource.username=root
spring.datasource.password=your_password
