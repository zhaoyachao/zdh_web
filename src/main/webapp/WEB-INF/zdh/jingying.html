功能模块一览

覆盖数据全生命周期的完整功能矩阵

数据集成中心

  • DataX引擎：基于DataX的离线数据同步，支持200+数据源
  • Flink CDC：实时变更数据捕获，支持MySQL/PostgreSQL/Oracle
  • Kafka集成：高吞吐消息队列，支持流批一体处理
  • API接口：RESTful API数据接入，支持第三方系统对接
  • 文件采集：日志文件、CSV/Excel、FTP/SFTP自动采集

任务调度中心

  • DAG工作流：可视化拖拽编排，支持复杂依赖关系
  • 定时调度：Cron表达式、手动触发、事件驱动
  • 分布式执行：支持多节点并行执行，负载均衡
  • 失败重试：智能重试策略、告警通知、断点续跑
  • 资源管控：CPU/内存限制、队列优先级、并发控制

数据开发环境

  • SQL IDE：在线SQL编辑器，语法高亮、智能补全、格式化
  • UDF管理：自定义函数开发、测试、发布一体化
  • 版本控制：基于Git的代码版本管理，支持回滚
  • 代码审核：Code Review流程，保障代码质量
  • 调试工具：断点调试、性能分析、执行计划查看

数据质量管理

  • 规则配置：内置100+质量规则，支持自定义规则扩展
  • 异常检测：统计方法+机器学习，自动识别数据异常
  • 质量报告：DQ评分、趋势分析、问题根因定位
  • 数据清洗：ETL清洗流程、去重、标准化、脱敏
  • 血缘追踪：字段级血缘关系、影响分析、溯源查询

快速开始

三步启动您的数字化经营平台

Step 1: 获取代码

从GitHub或Gitee克隆项目代码到本地

git clone https://github.com/zhaoyachao/zdh_web.git

Step 2: 编译打包

使用Maven进行项目编译和打包

sh build.sh

Step 3: 启动应用

启动应用并访问数字化经营平台

sh bin/start.sh
# 访问 http://localhost:8081

配置说明

在启动前，请确保在 application.properties 中配置以下参数：

# 数据库配置
spring.datasource.url=jdbc:mysql://localhost:3306/zdh
spring.datasource.username=root
spring.datasource.password=your_password

# Redis配置（缓存和任务队列）
spring.redis.host=localhost
spring.redis.port=6379

# 数据源配置（可选）
etl.datax.home=/opt/datax
etl.flink.home=/opt/flink
